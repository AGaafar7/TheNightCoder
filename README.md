## 🚀 Getting Started

### 1. Install the Plugin

- Open **JetBrains Fleet**
- Go to **Plugins**
- Search for **Red & Black Fleet Theme**
- Click **Install**

### 2. Apply the Theme

- Open **Settings**
- Navigate to **Appearance > Theme**
- Select **Red & Black Fleet Theme**

You're now rocking a sleek and powerful new look! 😎
